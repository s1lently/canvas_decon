# func/ 目录重命名计划

## 命名规范分析
当前func/已有良好的getXxx命名，需要补充规范：
- getXxx.py  - 获取/抓取数据
- procXxx.py - 处理/加工数据  
- mgrXxx.py  - 管理器
- utilXxx.py - 工具函数

## 重命名方案

### 保持不变 (已符合getXxx规范, 9个)
func/getCourses.py          → 保持
func/getHistoryTodos.py     → 保持
func/getHomework.py         → 保持
func/getQuiz_ultra.py       → 保持
func/getSyll.py             → 保持
func/getTodos.py            → 保持

### 需要重命名 (6个)

func/learn_material.py          → func/procLearnMaterial.py (处理学习材料)
func/model_selector.py          → func/utilModelSelector.py (工具-模型选择)
func/pdf_bookmark_extractor.py  → func/utilPdfBookmark.py   (工具-PDF书签)
func/pdf_splitter.py            → func/utilPdfSplitter.py   (工具-PDF拆分)
func/upPromptFiles.py           → func/utilPromptFiles.py   (工具-Prompt上传)
